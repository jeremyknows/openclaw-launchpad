Questions?

Is this safe?

Yes. ClawStarter sets up OpenClaw to run locally on your Mac. Your data stays on your machine â€” no cloud middleman.

The setup script uses industry-standard security practices. Your API keys are stored locally and only sent to the AI provider you choose (OpenAI, Anthropic, etc.). Nothing is transmitted to third parties.

Do I need to know how to code?

Nope. If you can copy and paste, you can run ClawStarter.

Paste one command into Terminal, press Enter, and answer 2 simple questions. The script handles everything else. Takes about 2 minutes.

What can it actually do?

Out of the box, your AI can:

  • Answer questions and have conversations
  • Search the web for real-time info
  • Remember context from past chats
  • Read and write files
  • Run terminal commands (with your permission)

You can connect messaging apps (Discord, iMessage, Telegram) and add more capabilities later.

What’s the catch?

There isn’t one. This is an open-source project built for real people who want a personal AI without vendor lock-in.

You own your data. You control the costs. You decide what it can access. No subscriptions, no upsells, no tracking.

My Mac goes to sleep. Will this work?

The setup guide includes a command to keep your Mac awake (but turns off the screen to save power). This ensures your AI is always responsive.

If you need your Mac to sleep (like a laptop you carry around), you can skip that step — just know your AI won’t respond while the Mac is asleep.

Can I use this on multiple Macs?

Yes! Run ClawStarter on each Mac. Each will have its own independent AI agent. They don’t sync automatically, but you can share config files between them if you want.

What if I want to uninstall it?

Delete the bot user account from System Settings → Users & Groups. That’s it. Everything is contained in that one user account.
